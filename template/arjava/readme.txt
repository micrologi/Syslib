# Minimal Requirement
----------------------------------------------------
- Slims Application
- Browser with CSS3 and HTML5 support
- Pation and pation

----------------------------------------------------
# Installation
----------------------------------------------------
1. Extract zip file to template folder
2. Log in to your back end system
3. Click on System menu
4. Select 'Arjava' template for you Public Template
5. Try it now by click on OPAC menu

----------------------------------------------------
# Footer Information (display on footer)
----------------------------------------------------
1. Log in to your back end system
2. Click on System menu
3. Click on Content, create 3 new content with this unique path** : membership, contact, opening 
   Or you can use sample-content.sql (included) for sample data. Please import used phpMyAdmin
   
   **If you use different name for your path, please change it on index_template.inc inside template folder (on line 130)
   $path = array('membership','opening', 'contact'); /* change as needed */

----------------------------------------------------
